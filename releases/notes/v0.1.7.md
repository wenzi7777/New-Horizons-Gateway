# New Horizons Gateway v0.1.7

## Changes

- Fixed UDP command dispatch: `udp_commands.set_session` for binary heartbeat and stream packets now records the device's source address directly (`addr`) instead of substituting `CONTROL_PORT` (22345). The device's UDP command socket is `kUdpStreamPort` (13250) — the same socket used to send heartbeat and stream data — so commands must target that port. `CONTROL_PORT` (22345) is a TCP-only constant and must not be used for UDP sessions.
