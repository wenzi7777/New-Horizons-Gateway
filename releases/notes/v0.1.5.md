# New Horizons Gateway v0.1.5

## Changes

- Fixed UDP command dispatch: `udp_commands.set_session` is now populated from heartbeat and stream packets so commands are always routed over the active UDP path instead of failing silently.
- Added result field normalization in `handle_udp_control`: `cmd` → `command` and `ok` → `status: "ok"/"error"` to match the Desktop backend's expected schema.
- Replaced all legacy startup scripts with clean cross-platform scripts:
  - `scripts/start.sh` / `scripts/stop.sh` — host-mode (Mac/Linux)
  - `scripts/start_docker.sh` / `scripts/stop_docker.sh` — Docker (Mac/Linux)
  - `scripts/start.ps1` / `scripts/stop.ps1` — host-mode (Windows)
  - `scripts/start_docker.ps1` / `scripts/stop_docker.ps1` — Docker (Windows)
