# New Horizons Gateway v0.3.1

## Highlights

- Added a simple Windows console TUI for the native Gateway launcher. The top
  area now stays fixed with live Gateway status, while recent events continue
  scrolling below.
- High-frequency `/api/status` polling no longer floods the dedicated Gateway
  console. Polls are summarized as a counter in the TUI header instead.
- The Gateway WebUI `Update Center` now turns green when the current version is
  allowed to run and no update is required, making the healthy state clearer.
