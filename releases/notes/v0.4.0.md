# New Horizons Gateway v0.4.0

## Highlights

- Reworked Gateway OTA into a staged `current -> next -> restart activate`
  flow. The running Gateway no longer tries to replace its own live
  `scripts/` and package directories in-place during `apply`.
- Fixed the Windows update stall where `apply` could appear stuck around
  `68%` while touching the live launcher tree. Updates are now prepared in a
  pending release directory and activated only during restart.
- Simplified the Gateway Web UI update section to a single progress bar so the
  OTA state is easier to understand during mandatory updates.
