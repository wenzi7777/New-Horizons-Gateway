# New Horizons Gateway v0.1.8

## Changes

- Removed unused TCP control code: `send_control_command` (arduino_protocol.py), `arduino_sessions` dict (main.py), `CONTROL_PORT` import (main.py), and `tcp_server` parameter (web.py). The gateway has been UDP-only since v0.1.6; these were dead code.
- Fixed FindMe multi-gateway transfer: when a device's discover packet carries `preferred_gateway_id` pointing to a different gateway, this gateway now responds with `accept: false` instead of a normal offer. This prevents the previous gateway from competing and causing oscillation during a "Serve This Device" transfer.
