# New Horizons Gateway v0.3.2

## Highlights

- Fixed the OTA apply path so Gateway updates now complete as a real
  `check -> download -> apply` workflow instead of stalling before restart.
- Added one-click `Update now` flow in the Gateway Web UI. When an update is
  required, Gateway now downloads and applies the release automatically, then
  leaves the user only the final restart step.
- Added visible progress bars for both `Download` and `Apply`, including the
  forced-update overlay, so update progress is no longer opaque.
- Kept the new healthy-state visuals: when the current Gateway version is
  allowed to run, the `Update Center` remains in its green status style.
