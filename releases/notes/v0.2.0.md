# New Horizons Gateway v0.2.0

## Changes

- Complete WebUI overhaul: sticky topbar with live upstream badge, cleaner card layout, toggle-switch for gateway enable/disable, and Traditional Chinese (繁體中文) added as third language.
- Three-step setup wizard (Gateway ID → upstream server → confirm & enable) replaces the single-field first-run prompt, making initial configuration self-explanatory.
- Nearby devices panel is now visible by default so multi-gateway LAN setups immediately show all devices seen via FindMe broadcasts — no manual "Discover" click needed.
- "Serve this device" button renamed to "Transfer to this gateway" with a clear disabled-state tooltip when upstream is offline, reducing confusion in multi-gateway deployments.
- Docker start scripts always rebuild the image on start (`--build` flag always passed), so code changes are picked up without a separate manual rebuild step.
